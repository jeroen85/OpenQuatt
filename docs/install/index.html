Installatiehulp

Verbind of installeer je OpenQuatt-module

Stel Wi-Fi in op een voorgeïnstalleerde module of flash zelf de stabiele firmware.

Let op OpenQuatt kan gevolgen hebben voor je Quatt-garantie.

Voorgeïnstalleerde module

Alleen Wi-Fi instellen

Gebruik dit als je module al met OpenQuatt is geleverd. Deze stap zet alleen je Wi-Fi-gegevens op de module en flasht niets.

  1. Sluit de module met USB aan op deze computer.
  2. Klik op Configureer Wi-Fi en kies de juiste USB-poort.
  3. Vul je Wi-Fi-netwerk in en open daarna http://openquatt.local.

Sluit je voorgeïnstalleerde module via USB aan om Wi-Fi te configureren.

Open OpenQuatt

Zelf installeren

1

Kies je opstelling

Hiermee bepaal je of de firmware als Single of Duo wordt opgebouwd.

2

Kies je ESP-module

Hiermee bepaal je de exacte bordindeling en het juiste factory-image.

De Heatpump Controller Q-edition is verkrijgbaar via de productpagina van Electropaultje.

3

Kies je verbinding

De installer activeert alleen verbindingen waarvoor in de stabiele release een factory-image beschikbaar is.

Handmatige route

Open de nieuwste stabiele release Flash een gedownloade factory binary in ESP Web Tools

Daarna

Netwerk afronden

Bij Wi-Fi stel je het netwerk via USB in. Bij Ethernet sluit je na het flashen de netwerkkabel aan.

Web-app openen

Open daarna http://openquatt.local en rond de Quick Start af voordat je verdergaat in Home Assistant.

Fallback route

Lukt Wi-Fi instellen via USB niet, verbind dan met het OpenQuatt access point en stel Wi-Fi handmatig in. Ethernet-builds zijn Ethernet-only.
