OpenQuatt / Stabiele release

Installeer de nieuwste stabiele firmware zonder zelf binaries te zoeken.

Kies je opstelling en hardwareprofiel en flash daarna direct het juiste OpenQuatt factory-image via USB. OTA-updates blijven na de eerste start gewoon via de bestaande GitHub Release-flow lopen.

Gebruik Chrome of Edge op een desktop of laptop.

Sluit een USB-datakabel aan, niet alleen een voedingskabel.

Laat deze pagina na het flashen open zodat de browser wifi-instelling via USB kan aanbieden.

Lukt dat niet, dan blijft het fallback access point van OpenQuatt beschikbaar.
