# MQTT verwijderd

MQTT is uit de standaard OpenQuatt firmware gehaald. De experimentele publish-only export werkte onvoldoende betrouwbaar en trok via ESPHome native MQTT honderden extra per-entity MQTT-componenten de firmware in.

Gebruik voor normale integratie Home Assistant via de native ESPHome API.
